# 支払い元JSON 作成テンプレート（生成AI向け）

以下の条件で、家計簿アプリにインポートするためのJSONを作成してください。
出力は **JSONのみ** にしてください。説明文は不要です。

## 目的
- 支払い元（現金、銀行、プリペイド等）をまとめて登録したい

## 出力フォーマット
```json
{
  "accounts": [
    {
      "name": "支払い元名",
      "kind": "cash|bank|prepaid|other|card",
      "note": "任意"
    }
  ]
}
```

## ルール
- `name` は必須。重複させない
- `kind` は次のいずれかを使う: `cash`, `bank`, `prepaid`, `other`, `card`
- `note` は任意
- 余計なキーは含めない

## 入力情報（あなたが埋める）
- 現金系:
- 銀行口座系:
- プリペイド系（Suica, PayPayなど）:
- クレジットカード系:
- その他:

## 期待する出力
- 上記情報を `accounts` 配列にしてJSONのみ出力
